# BlindAIAgent
在Windows环境下，通过无障碍辅助功能，为仅文本的单模态大模型提供GUI操作能力

# 原理
基于Windows系统中，通过[UI自动化](https://learn.microsoft.com/zh-cn/dotnet/framework/ui-automation/ui-automation-overview)功能，将屏幕上的所有控件转换为文本并编号。大模型根据任务，给出对某控件的某操作，由此工程执行。
